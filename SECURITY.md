# Security Policy

## Reporting a Vulnerability

Please report vulnerabilities privately to Florin Badita before opening a public issue.

Disclosure email: florin.badita@gmail.com

Include a concise description, reproduction steps, affected browser or platform, and any relevant files or screenshots.

## Baseline

- No secrets are stored in the frontend.
- The app runs fully client-side on GitHub Pages.
- Uploaded images stay in the browser unless a user explicitly exports or shares them.
- Local persistence uses browser storage only.
